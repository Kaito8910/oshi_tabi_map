# 推し旅マップ

推し活のイベント・グッズ・費用・写真をまとめて記録できるFlutter製スマホアプリです。

## アプリ概要

推し活で参加したイベントや購入したグッズ、遠征費などを一括で管理できます。
イベントごとの写真記録や、推しごとのグッズ管理にも対応しています。

## 主な機能

### 推し管理

* 推し情報の登録
* グループ名・メンバーカラー設定
* 推しメモ管理

### イベント管理

* イベント登録
* 日付・会場管理
* 次回イベント表示
* 過去イベント一覧表示

### グッズ管理

* グッズ登録
* カテゴリ管理
* 購入数・金額管理
* イベント別グッズ管理
* 推し別グッズ管理

### 費用管理

* チケット代
* 交通費
* 宿泊費
* 飲食代
* その他費用管理

### 写真管理

* イベント写真登録
* アルバム表示
* 写真拡大表示

### バックアップ機能

* ZIP形式バックアップ
* データ復元
* データ初期化

### その他

* ダークモード
* 金額非表示機能
* 遠征都道府県数表示
